The Model Targets sample shows how to detect a 3D object and render a simple 3D representation on top of it.

Key Functionality

  • Load and activate a single Model target database at a time
  • Detection using the initial Detection Position of the 3D Model target
  • Automatic 3D object tracking after successful detection
  • Manage camera functions: continuous autofocus

Instructions

  • Point camera at target to view
  • Single tap to focus
  • Double tap to access options menu
